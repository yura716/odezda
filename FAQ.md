# ❓ Часто задаваемые вопросы (FAQ)

## Общие вопросы

### Что такое Odezda AI?

Odezda AI - это веб-приложение, которое использует искусственный интеллект для анализа вашей внешности и подбора одежды в желаемом стиле. Вы загружаете свое фото, указываете стиль, и получаете персональные рекомендации с ссылками на магазины.

### Какие технологии используются?

- **Frontend**: React + современный CSS
- **Backend**: Python FastAPI
- **AI**: OpenAI GPT-4o (Vision API)
- **Поиск товаров**: интеграция с API магазинов

### Бесплатно ли это?

Само приложение бесплатное и с открытым исходным кодом. Однако для работы требуется OpenAI API ключ, который является платным (примерно $0.01-0.03 за запрос).

## Установка и настройка

### Где получить OpenAI API ключ?

1. Зарегистрируйтесь на https://platform.openai.com/
2. Перейдите в раздел "API keys"
3. Нажмите "Create new secret key"
4. Скопируйте ключ и добавьте в `backend/.env`

⚠️ **Важно**: Никому не показывайте ваш API ключ!

### Почему backend не запускается?

Проверьте:
1. Установлен ли Python 3.8+ (`python3 --version`)
2. Установлены ли зависимости (`pip list`)
3. Правильно ли указан API ключ в `backend/.env`
4. Свободен ли порт 8000 (`lsof -i :8000`)

### Почему frontend не может подключиться к backend?

Убедитесь что:
1. Backend запущен и работает (откройте http://localhost:8000)
2. В `frontend/.env` указан правильный URL backend
3. В консоли браузера (F12) нет ошибок CORS

## Использование

### Какие форматы фото поддерживаются?

Поддерживаются: **JPG, PNG, GIF, WebP**

Ограничения:
- Максимальный размер: 10 MB
- Рекомендуемое разрешение: 1024x1024 или меньше

### Какие стили одежды доступны?

По умолчанию доступны:
- Casual (повседневный)
- Business (деловой)
- Sport (спортивный)
- Street style (уличный)
- Elegant (элегантный)
- Romantic (романтичный)
- Boho (бохо)
- Minimalist (минималистичный)
- Vintage (винтаж)
- Gothic (готический)

Вы также можете описать свой собственный стиль в текстовом поле!

### Насколько точны рекомендации?

Рекомендации основаны на анализе GPT-4 Vision, одной из самых продвинутых AI моделей. Однако помните, что это все еще AI, и рекомендации могут быть субъективными. Используйте их как вдохновение и адаптируйте под свой вкус.

### Сохраняются ли мои фото?

По умолчанию фото **не сохраняются** на сервере. Они обрабатываются в памяти и сразу удаляются. Фото отправляется только в OpenAI API для анализа согласно их [политике конфиденциальности](https://openai.com/policies/privacy-policy).

## Разработка

### Как изменить промпт для AI?

Откройте `backend/main.py` и найдите функцию `analyze_image_and_style()`. Там вы можете изменить переменную `prompt` для более детального анализа.

Примеры улучшений:
- Добавить анализ цветотипа
- Учитывать сезон
- Учитывать бюджет
- Добавить анализ пропорций тела

### Как добавить базу данных?

Для добавления базы данных:
1. Установите SQLAlchemy или другую ORM
2. Создайте модели для пользователей и их запросов
3. Добавьте эндпоинты для сохранения/загрузки истории

Пример:
```python
from sqlalchemy import create_engine, Column, Integer, String
from sqlalchemy.ext.declarative import declarative_base

Base = declarative_base()

class UserQuery(Base):
    __tablename__ = 'queries'
    id = Column(Integer, primary_key=True)
    user_id = Column(String)
    style = Column(String)
    results = Column(JSON)
```

### Как добавить аутентификацию?

Используйте JWT токены:
1. Установите `python-jose` и `passlib`
2. Создайте эндпоинты для регистрации/логина
3. Защитите `/api/analyze` с помощью зависимостей FastAPI

## Деплой

### Где лучше задеплоить приложение?

**Backend:**
- Railway (самый простой)
- Render
- DigitalOcean App Platform
- Heroku

**Frontend:**
- Vercel (рекомендуется)
- Netlify
- GitHub Pages

### Нужно ли что-то менять для деплоя?

Да, обязательно:
1. Измените `ALLOWED_ORIGINS` в `backend/.env` на URL вашего frontend
2. Добавьте переменную окружения `OPENAI_API_KEY` в настройках хостинга
3. Обновите `REACT_APP_API_URL` в frontend на URL вашего backend

### Сколько стоит запустить в продакшене?

Примерные затраты:
- **Хостинг backend**: $5-10/месяц (Railway, Render)
- **Хостинг frontend**: бесплатно (Vercel, Netlify)
- **OpenAI API**: $0.01-0.03 за запрос
- **Домен**: $10-15/год

При 1000 запросов в месяц: ~$25-40/месяц

## Производительность

### Почему анализ занимает много времени?

Анализ обычно занимает 5-15 секунд из-за:
1. Загрузки изображения на сервер
2. Обработки и сжатия изображения
3. Запроса к OpenAI API (самая долгая часть)
4. Генерации ссылок на товары

Это нормально для AI-анализа.

### Можно ли ускорить работу?

Да, несколько способов:
1. **Кеширование**: сохраняйте результаты для одинаковых запросов
2. **Оптимизация изображений**: сжимайте на клиенте перед отправкой
3. **CDN**: используйте CDN для статических файлов
4. **Очередь задач**: используйте Celery для асинхронной обработки

## Безопасность

### Безопасно ли загружать свои фото?

Приложение не хранит ваши фото, но они отправляются в OpenAI API. Изучите [политику OpenAI](https://openai.com/policies/privacy-policy) перед использованием.

Для максимальной безопасности:
- Запускайте локально на своем компьютере
- Не используйте фото с паспортными данными
- Не загружайте фото детей без согласия родителей

### Как защитить API от злоупотреблений?

Добавьте:
1. **Rate limiting**: ограничение количества запросов
2. **Аутентификацию**: только зарегистрированные пользователи
3. **Мониторинг**: логирование всех запросов
4. **Валидацию**: проверка размера и формата файлов

Пример rate limiting:
```python
from slowapi import Limiter
from slowapi.util import get_remote_address

limiter = Limiter(key_func=get_remote_address)

@app.post("/api/analyze")
@limiter.limit("10/minute")
async def analyze_photo(...):
    ...
```
